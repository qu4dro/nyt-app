## New York Times app
A news android app with the ability to display new popular news, save your reading list and search for specific news on demand.

[Api](https://developer.nytimes.com/apis)

## Technologies
* Kotlin
* Architecture components
* MVVM
* Navigation
* Coroutines
* Room
* Retrofit + Moshi
* Dagger hilt
* Recycler view
* Data binding
* LiveData + Flow
* Coil
* Material components
